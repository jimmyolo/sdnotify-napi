# sdnotify-napi

> another sd-notify native binding, but in N-API

## Requirements

- systemd support
- node.js >= 16.0
- systemd dev files
  - debian: `apt install libsystemd-dev`
  - redhat: `yum install systemd-devel`

## Usage

same as [`sd-notify`](https://github.com/systemd/node-sd-notify)

- [ ] ready
- [ ] stopping
- [ ] watchdog
- [ ] watchdogInterval
- [ ] startWatchdogMode
- [ ] stopWatchdogMode
- [ ] sendState
- [ ] journalPrint
